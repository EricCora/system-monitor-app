# PulseBar Icon Candidates

These are 1024x1024 SVG concepts for the app icon.

Files:
- `pulsebar-icon-bars.svg`: stacked telemetry bars in a warm glass tile
- `pulsebar-icon-orbit.svg`: circular pulse/orbit meter
- `pulsebar-icon-grid.svg`: dashboard panel grid with active pulse line
- `pulsebar-icon-thermal.svg`: temperature-focused glyph with monitor bars

Open them in Finder, Quick Look, or a browser to compare.
